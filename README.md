# chgnet-web

##这是一个基于vue框架的web网页，后端文件为chgnet_reload,利用flask传递文件，可以接收物质结构的cif文件并进行一些分析。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
